# Tarea-N-03: Heladería y Dulces Regionales "Doña Solina"

Este proyecto consiste en un sistema de gestión para una pequeña empresa de dulces regionales y helados. Incluye control de usuarios, clientes, productos y pedidos.

## Características
- **Sistema de Login**: Acceso restringido mediante usuarios registrados en la base de datos.
- **Dashboard**: Vista general con estadísticas rápidas.
- **Gestión de Clientes**: Registro y visualización de clientes.
- **Inventario**: Control de productos regionales, categorías, regiones y stock.
- **Pedidos**: Registro de compras vinculando clientes y productos.

## Tecnologías Utilizadas
- **PHP**: Lógica del servidor y conexión a base de datos.
- **MySQL**: Base de datos `TareaN°03`.
- **Bootstrap 5**: Interfaz de usuario moderna y responsiva.
- **Git**: Control de versiones con flujo de ramas.

## Requisitos
1. Servidor local (XAMPP, WAMP, etc.).
2. Base de datos MySQL con el nombre `TareaN°03`.
3. Tablas: `usuario`, `cliente`, `producto_regional`, `pedido`.
